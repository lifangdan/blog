    PS简单操作

    vuePress-theme-reco 非洲猫    2019 - 2023

    PS简单操作

    非洲猫 12/17/2020 PS

    # 一、工具

    • 1、减淡工具、加深工具可以给图片上阴影，比如人物脸部加上后后显得更加立体。
    • 2、滤镜下的液化功能可以p瘦等等。
    • 3、蒙版可以让两张图片柔和重叠在一起，适用于图层混合。
    • 4、把A图移到B图上时，按shift键拖过来就行了。
    • 5、shift+D取消选区。
    • 6、ctrl+T，选中图片，按shift键可以等比例缩小图片； bridge软件与ps配合使用,可以保留图片原图,不会让图片有损失

    # 二、调色（图像-->调整）

    按住Alt键就会出现‘复位’按钮

    • 1、色阶： 移动输入色阶； 吸管工具吸取亮部和暗部； 自动色阶（打开图片后Ctrl+shift+L）；
    • 2、曲线
    • 3、色相饱和度： 可借用选区工具做大概选择； 吸管工具吸取颜色；
    • 4、色彩平衡：（主要调冷色和暖色）；
    • 5、匹配颜色： 获取好片子的光源，让自己拍暗的、没有颜色的片子有好的曝光； 适用于将两张图合在一起时，将其互相匹配；
    • 6、调灰色： 通道混合器（效果最好）； 灰度模式（效果次好）； 去色；
    • 7、渐变映射：
    • 8、照片滤镜： 人物用暖色滤镜； 风景用冷色滤镜；
    • 9、阴影/高光： 适用于那些暗角看不到细节的图片； 逆光拍出的图片；
    • 10、阈值： 使用阈值之前先使用‘滤镜->其他->高反差保留';

    # 三、通道案例

    把火苗（图A）抠到另一张图（图B）上

    # 方法一：

    把图A拉到图B上，然后将图A设置成正片叠底模式；

    # 方法二：

    • 1、首先把图A的三个通道值分别拷贝出来（按ctrl键选中红绿蓝通道，拖动到下方的新建通道即可拷贝）；
    • 2、然后按ctrl键选中红色通道拷贝副本，选区选中火苗后，返回到图层，新建图层，颜色调成红色，按alt+delete将选中的红色通道粘贴到图层上；
    • 3、绿色通道和蓝色通道也是一样操作；
    • 4、然后将红色和绿色图层设置成‘滤色’，滤色是加色，正片叠底是减色；
    • 5、选中三个图层，按ctrl+E，将三个图层合成一个图层；
    • 6、然后直接鼠标选中该图层拖到图B上即可完成；

    # 四、图层

    • 1、图层混合 把两个图层混合在一起， 图层->图层样式->混合选项->调节'混合颜色带'; 按住Alt键拆开三角形,调节容差值;
    • 2、 高级混合 双击图层,挖空设置成‘深’，填充不透明度设置适当降低，则该图层会穿过中间图层，见到最底下的图层内容。
    • 3、分离图层样式
    • 图层->图层样式->创建图层,即可将该图层所具有的图层样式创建出来;例如投影,可以Ctrl+T,随意调节投影形状;
    • 4、新建填充图层 创建好后选择黑颜色画笔即可将不要的部分擦掉；
    • 5、新建调整图层 可调整图层的颜色、饱和度等等，并且不会破坏原片，跟蒙版类似。 若勾选‘使用前一图层创建剪贴蒙版’，则调整效果只对前一图层起作用；
    • 6、创建剪贴蒙版（编组）只适合大图放在小图上面使用 图层A与图层B，把图层B创建剪贴蒙版，即可将图层B的内容放到图层A的形状上。 也可以直接按住Alt,然后鼠标放到图层B上,出现不一样形状时,点击一下即可完成编组。
    • 7、制作人在草丛中跳舞 把人用选区选中，然后建立工作路径； 然后在图层->尺量蒙版->当前路径; 然后再创建蒙版,用画笔工具,将一些草画出来,适当盖住人,这样就制作出人在草丛中跳舞的场景。

    # 五、将一个斜的图校正

    选择‘透视裁切工具‘，画一个框，然后把框移到图片一角，使重合，然后把画的框的中心点移到那个角，然后把框与图片拉到重合，按回车键即可校正

    # 六、滤镜

    • 1、将图片背景模糊 打开图片，在通道新建一个alpha通道； 用渐变工具在alpha通道画一个区域，白色部分将是被模糊的部分； 回到图层，选择滤镜->模糊->镜头模糊; '源'选项选择新建的alpha通道,调整模糊焦距即可改变模糊区域;

    # 七、调色

    • 1、做自动色阶
    • 2、调饱和度
    • 3、调色，调色彩平衡
    • 4、锐化（为了让轮廓更清楚） 把背景图复制一个图层，然后选择滤镜-其他-高反差保留，调整半径，一般调为一两个像素能看到轮廓就差不多， 然后选择柔光模式， 如果想看到更清晰轮廓，可以再复制多一个已经做好的锐化图层

    # 八、画菊花，Ctrl+T的使用，可制作画中画的效果

    • 1.画个椭圆，ctrl+t，做旋转一定的角度；
    • 2、ctrl+shift+t，即可重复你上一个旋转的动作；
    • 3、ctrl+shift+alt+t，即可复制粘贴你上一个动作，这样菊花就画成了；
