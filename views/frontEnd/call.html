    call()、apply()、bind() 的用法

    非洲猫 9/19/2021 前端


    如果你常常分不清这三者的用法和区别，那你需要好好看看这篇文章啦~
    这三个函数其实大同小异，总的目的就是去指定一个函数的上下文（this）
    请看下面的例子：

    let obj = {
  name: 'feizhoumao',
  fun: function () {
    console.log(this);
  }
};
obj.fun()

    上面代码打印结果都是一样的：{ name: 'feizhoumao', fun: [Function: fun] }，说明此时函数内的 this 指向的是obj这个对象。
    再看下面的代码：

    let obj = {
  name: 'feizhoumao',
  fun: function () {
    console.log(this);
  }
}
let obj2 = {
  name: 'lifangdan'
};
obj.fun.call(obj2);
obj.fun.apply(obj2);
obj.fun.bind(obj2)();

    上面代码打印结果是：{ name: 'lifangdan' }，说明此时函数内的 this 指向了 obj2 这个对象。值得注意的是，bind方法后面多了个 () ，因为bind 返回的是一个新的函数，你必须调用它才会被执行。

    # 传参区别

    接下来我们看看这三个函数传参的区别。

    let obj = {
  name: 'feizhoumao',
  age: 18,
  fun: function (i, j) {
    console.log(this.name + '，今年' + this.age + '岁，来自' + i + '，喜欢' + j + '！');
  }
}
let obj2 = {
  name: 'lifangdan',
  age: 20
};
obj.fun('广西', '编程')
obj.fun.call(obj2, '广西', '编程');
obj.fun.apply(obj2, ['广西', '编程']);
obj.fun.bind(obj2, '广西', '编程')();

//打印结果分别如下：
//feizhoumao，今年18岁，来自广西，喜欢编程！
//lifangdan，今年20岁，来自广西，喜欢编程！
//lifangdan，今年20岁，来自广西，喜欢编程！
//lifangdan，今年20岁，来自广西，喜欢编程！

    不难看出:
    call 、bind 、 apply 这三个函数的第一个参数都是 this 的指向对象，第二个参数差别就来了：
    call的参数是直接放进去的全都用逗号分隔。
    apply的所有参数都必须放在一个数组里面传进去。
    bind除了返回是函数以外，它的参数和call一样。
    当然，三者的参数不限定是 string 类型，允许是各种类型，包括函数 、 object 等！
